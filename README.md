# 体脂肪計測アプリ

体脂肪率を記録して、変化をグラフで確認できるアプリです。

現在は開発の初期段階です。

## 機能の予定

- 体脂肪率・体重の日次記録
- 推移のグラフ表示(週・月・年)
- 目標値の設定と達成度の表示

## 記録するデータ(案)

| 項目 | 単位 | 備考 |
|---|---|---|
| 体脂肪率 | % | 必須 |
| 体重 | kg | 必須 |
| 測定日時 | - | 自動記録 |
| メモ | - | 任意(食事・運動など) |

## 公開URL

## 開発方法

Node.js(LTS)が必要です。

```bash
npm install   # 初回のみ: 依存パッケージのインストール
npm run dev   # 開発サーバー起動 → http://localhost:3000
npm test      # テスト実行
npm run build # 本番ビルド
```

詳しい計画は [docs/requirements.md](docs/requirements.md) を参照してください。
